# 🎖️ 儿童军棋 (Children's Chess)

一个简单有趣的儿童军棋游戏，使用纯HTML/CSS/JavaScript开发。

## 功能特性

- 🎮 交互式游戏界面
- 📱 响应式设计（支持手机和平板）
- 🎨 现代化的UI设计
- ⚡ 无需依赖，轻量级

## 游戏规则

- 选择棋子进行游戏
- 红方和蓝方轮流操作
- 每次操作获得分数

## 访问游戏

您可以直接在浏览器中打开 `index.html` 文件来玩游戏，或访问：

https://adelajin15-source.github.io/Adela/

## 技术栈

- HTML5
- CSS3（Grid 布局、渐变背景）
- Vanilla JavaScript（无框架）

## 开发建议

如果要扩展功能，可以将代码分离为：
- `style.css` - 样式文件
- `game.js` - 游戏逻辑
- `index.html` - HTML结构
